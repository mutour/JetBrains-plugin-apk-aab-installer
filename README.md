# ApkInstaller

A JetBrains IntelliJ plugin to install APK/AAB files to connected Android devices directly from the IDE.

## Features

- Install APK/AAB files with one click
- Multi-device support (select multiple devices)
- Background installation
- Automatic ADB detection
- Bundletool support for AAB files

## Installation

Download the latest release from the JetBrains Plugin Market or build from source.

## Usage

1. Right-click on an `.apk` or `.aab` file in the Project View or Editor Tab.
2. Select "Install to Device(s)".
3. Select target devices (if multiple connected).
4. Wait for installation to complete.

## Requirements

- IntelliJ IDEA 2024.1+ or Android Studio
- ADB installed and configured
- For AAB installation: bundletool
